# DSA Problems
Data Structure, Algorithms and Problem-Solving repository

---

## Quick Start

**Prerequisites:** Python 3

```bash
# 1. Clone repository
git clone git clone git@github.com:romaincroiz/dsa_problems.git
# 2. Create a virtual environment and import packages
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

```

## What is this?

This sandbox project provides implementations and explanations to better understand Data 
Structure and their use in DSA problems. For better portability, DSA solutions will use data structures 
already provided by the language.

## Structure

- [data_structure](./data_structure): Data Structure implementations
- [dsa](./dsa): Data Structure Algorithm problems and their solutions
